# Cool Climate Air Conditioning - Frontend

This is a Vite + React + Tailwind CSS frontend for an AC servicing website.

## Setup
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`

## Notes
- Fill company details in `src/data/siteData.js`.
- Replace placeholder images (e.g., `public/hero-ac.jpg`) with your own.
- Connect booking form to backend by updating `src/api/bookService.js`.# Cool-Climate-Air-Conditioning
